## alfred

1. 开启 Powerpack
2. 修改快捷键为刚才配的「F19」
![f19](https://clearmann.cn/uPic/GSM68n.jpg)
3. 把老电脑的 Alfred 配置复制到 ~/Documents/SoftwareConfiguration/Alfred 下，然后在「Advanced」里修改配置目录指向他，你的 Workflow 就全回来了
4. 「Features > Web Bookmarks」里记得把「Google Chrome Bookmarks」选上，这样就可以用 Alfred 模糊搜 Chrome 书签，用于快速打开网站。
5. 我们也可以在 Alfred 里边实现粘贴板记录，我习惯将展示粘贴板记录为 cmd + shift + c。
   ![设置](https://clearmann.cn/uPic/MxCCBo.jpg)
