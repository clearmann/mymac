# mymac
分享一下我的 MacBookPro 相关配置，喜欢的朋友希望可以推荐给更多的人！


## 快速开始

仓库里整理好了 `Brewfile`，安装好 [Homebrew](docs/homebrew.md) 后，一条命令即可装齐我常用的所有 App 与 CLI 工具：

```bash
brew bundle --file=./Brewfile
```
